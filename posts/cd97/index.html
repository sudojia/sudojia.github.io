不废话了，直接开始

安装环境

安装 Node.js，Git

并查看当前软件环境

配置 SSH key

查看 C:\Users\用户名\.ssh 下有无 ssh 文件夹，没有则创建

打开用户目录，找到 .ssh\id_rsa.pub 文件，用编辑器打开并复制里面的内容

然后打开你的 Github or Gitee 主页，进入个人设置 -> SSH公钥 -> 添加公钥
