别再打开网页搜 Maven 依赖了，试试这款 IDEA 插件

发表于2021-04-16|更新于2021-04-16|后端IntelliJ IDEA
|字数总计:269|阅读时长:1分钟

这款 IDEA 插件方便我们在 IDEA 中搜索依赖信息的工具，有时我们缺少依赖还得打开浏览器到 Maven 中央仓库找，而这款插件可以直接让你在 IDEA 开发工具查找。非常的快捷方便！

安装

在 IDEA 中打开 File | Settings | Plugins 搜索 search in repository 并安装

安装成功之后在右侧就会出现个 search 的工具栏，我们点击它，然后输入关键词，比如 spring-boot 并回车

可以看到，关于 spring-boot 的依赖信息都列出来了，此外我们还可以点击 Prev 和 Next 按钮切换分页。

接着双击 2.4.4 版本，然后 Maven 和 Gradle 的依赖信息都展现出来了，点击 Copy 即可复制粘贴到我们的 pom 文件中

对比 Maven 中央仓库，Usages 和 Date 都是一模一样

而且我们还可以搜 npm 包

结语

这款插件确实的快捷方便，但是这插件开发到现在还不足一个月，可能会遇到 BUG
