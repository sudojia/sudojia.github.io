Google Drive网盘直链分享、在线观影、加密保护

发表于2020-11-30|更新于2020-11-30|干货Google Drive
|字数总计:370|阅读时长:1分钟

有的小伙胖点进来不知道是什么，干嘛的，那你继续往下看

其实就是用 GoIndex 和 Cloudflare Workers 部署你的 Google Drive

不需要服务器和域名 Demo

这样你下载或者分享给人家，就可以不用挂那个东西了呀，直接下载

上传的话，可以用 GDIndex， 步骤是一模一样的

配合 IDM 下载还是挺快的

正式开始

准备工作

  1. Google 账号
  2. Cloudflare 账号

具体步骤

  1. 本文用的是一个 goindex 的扩展主题：goindex-theme-acrou

  2. 一键安装网址，打开以下任意网址（哪个打得开就用哪个）

    • https://goindex-quick-install.glitch.me
    • https://goindex-install.herokuapp.com
    • 如果你想要黑暗模式的话，可以用这个：https://goindex.glitch.me/

  3. 点进来之后，第一步就是授权，然后得到授权码并粘贴到 Authorization Code

  4. 填写表单，如果你想索引默认 15 G 那个盘的话，Driver ID 就填 root，团队盘的话，下面有讲怎么获取 Driver ID

Drive ID 获取

获取代码之后复制下来，并粘贴到 Cloudflare Workers 进行部署
