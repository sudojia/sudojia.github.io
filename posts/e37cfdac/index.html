使用 GitHub Actions 自动备份 Twikoo 评论数据

发表于2024-04-27|更新于2024-04-27|实用分享
|字数总计:533|阅读时长:2分钟

前言

也是心血来潮写的这个项目。因为在前面白嫖的 MongoDB 就被停用过，好在备份过一次，不然评论全无了

项目地址：

GitHub

https://github.com/sudojia/comment_backups

使用演示

  1. 【fork 项目】

  2. 在 comment_backups/settings/secrets/actions 填入所需变量

    变量名值
    PASSWORDTwikoo 密码
    TWIKOO_URL你的 Twikoo 地址

  3. 在 Settings -> Actions -> General 下给到读和写权限

  4. Run workflow 即可

  5. 如果 Actions 运行时没有出错，那么你的 Twikoo 评论数据就会出现在仓库内了

导入数据

  1. 先把仓库里的 twikoo-comment.json 下载到本地，然后去 【Json.cn】进行格式化

  2. 格式化之后按要求删除
