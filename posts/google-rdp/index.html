快速搭建永久免费Google RDP虚拟机

发表于2021-01-11|更新于2021-01-11|干货Google RDP
|字数总计:329|阅读时长:1分钟

首先在浏览器中打开无痕模式（ CTRL + SHIFT + N ）

进入 Colaboratory

注：要挂代理才能访问

Colaboratory

可在浏览器中编写和执行 Python 代码，并且

  • 无需任何配置
  • 免费使用 GPU
  • 轻松共享

然后左上角文件 –> 新建笔记本

登录 Google 账号

登录成功之后，复制以下代码并粘贴至输入框（直接 CTRL + V）

然后点击左边的启动按钮

安装过程中会出现个链接，让你复制 Debian Linux 命令并粘贴下来

所以我们就点击该链接

复制该命令

注：点一下才会出现输入框，然后粘贴进去，并按下 Enter 键

然后让你设置 6 位纯数字的 PIN 码

Enter 保存之后再输入一遍刚刚设置的 PIN 码

安装完成，我们点击该链接

点击该虚拟机

输入你刚刚设置的 PIN 码

进入到虚拟机桌面，我们点击使用默认配置就好了

可以看到此虚拟机是单独的 IP ，服务提供商是 Google Cloud

速度也是挺理想的

获取 root 权限

进入虚拟机桌面，右键 open Terminal Here

输入 sudo -i

默认密码是 8426
