国内服务器快速克隆 GitHub 开源项目

发表于2021-05-05|更新于2021-05-05|干货
|字数总计:355|阅读时长:1分钟

在 GitHub 发现个不错的开源项目，想放在服务器上跑，奈何服务器是华南地区，根本克隆不动，当然可以在本机下载下来传输到服务器上，但多此一举，我们可以在本机上借助 Google Chrome 上的扩展程序，安装一款叫 GitHub 加速 的插件！

安装

首先在本机安装此插件，地址一是谷歌扩展程序地址，地址二是为了照顾在墙内的人

下载地址 下载地址二

如果你是用地址二的方式，需要说明一下，下载成功之后，浏览器地址栏输入 chrome://extensions/，右上角把开发者模式打开，随后把下载好的 .zip 压缩包拖进去即可！

使用

打开 GitHub，随便找个开源项目，你就会发现多了个按钮

点击它，你就能看到不同的加速链接，有淘宝镜像源的，有 Cloudflare 反代的，还有 fastgit.org 的，一般来讲，淘宝的就足以了，其实这个插件就是整合了一些镜像源！

然后打开 SSH 工具连接到我们的服务器，先看下默认的克隆速度

你甚至可以打一把紧张刺激的 LOL 手游，一局下来也未必能克隆完成。

然后使用淘宝的镜像源进行克隆

一会儿的功夫就克隆好了！
