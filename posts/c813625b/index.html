使用 UnblockNeteaseMusic 解锁网易云音乐灰色无版权歌曲

发表于2021-09-22|更新于2021-09-22|干货网易云
|字数总计:564|阅读时长:1分钟

写在前面

UnblockNeteaseMusic，它是一款可以在网易云音乐里面能听到无版权的歌曲，说白了就是解锁网易云音乐客户端变灰的歌曲。（给这个灰色的歌曲换了个音乐源）

使用了 QQ / 虾米 / 百度 / 酷狗 / 酷我 / 咪咕 / JOOX 音源替换变灰歌曲链接（默认仅启用一、五、六）

前提准备

  1. 一台服务器 / VPS
  2. 网易云音乐客户端

开始安装

UnblockNeteaseMusic 的安装方式有很多，而本文呢，就讲我个人最喜欢的一种方式，使用的是 Docker 安装的方式。

安装 Docker

运行镜像

我们可以先直接使用 docker run 命令，不用先 docker pull 镜像，因为在 docker run 的时候他会去找这个镜像，发现没有它自己会进行一个 docker pull 的操作！

–name：容器名，看你喜欢叫什么，2333 则是本地端口号（指定一个本地的端口来映射容器内部的端口）自行修改，8080 是容器内部端口，无需更改。

接着我们输入 docker ps 查看当前正在运行的容器

最后放行端口！

使用

PC：

  1. 打开网易云音乐客户端，打开设置
  2. 找到工具，选择自定义代理，输入服务器 IP 地址和端口即可，点击测试，弹出【该代理可使用】
  3. 恭喜你，已经解锁网易云无版权歌曲了
  4. 最后点击【确定】重启网易云音乐客户端！

Android：

  1. 打开手机设置，或者长按 WIFI 图标即可进入
  2. 选择已连接的 WIFI 对它进行一个高级设置
  3. 代理选择手动，然后输入服务器 IP 和端口号即可
