利用Freenom免费注册的域名创建永久免费企业邮

发表于2021-01-12|更新于2021-01-12|干货
|字数总计:898|阅读时长:2分钟

在上篇文章中讲到

利用 Google RDP免费注册 Freenom 域名

https://blog.imzjw.cn/posts/rdp-freenom/

相信有不少小伙胖都注册成功了呢

那么本篇文章就 利用Freenom免费注册的域名创建永久免费企业邮

前言

目前还有免费的国外稳定可靠的域名邮箱有俄罗斯的 Yandex

还有美国的 Zoho，以及 25Mail.St ，Postale。

目前 Zoho 免费用户不支持 POP 和 SMTP

不过 Postale 和 Yandex 都支持 POP3 和 SMTP

国内的话….阿里云企业邮只能添加 50 个邮箱账号，腾讯企业邮和 163 的我没用过

如果是个人简单的域名邮箱日常使用，建议直接使用 Postale、腾讯企业邮或 163 和阿里云企业邮就可以了

这里我选择了俄罗斯的 Yandex，因为他是俄罗斯的啊，俄罗斯的 sister 又白又嫩

害！其实是因为 Yandex 邮箱一个域名下可以有 1000 个邮箱账号

前提

当然是一个域名和挂代理啦

正式开始

首先我们点击下面卡片链接进入到 Yandex Mail

Yandex-Mail

https://mail.yandex.com/

创号

填写信息，最后的电话号码可以不填，点 I don't have a mobile phone（我没有手机耶）

然后就会让你设置个密保问题，完事之后输入验证码点击注册

最后你就来到了这个界面

点击下面卡片链接进入到 Yandex Connect

Yandex Connect

https://connect.yandex.com/

点击右上角的 Try out，输入您的公司名称，随便输入。比如 imlsp

后面就直接看图吧！！

OK，点击 Add a domain

输入您的域名，比如我在 Freenom 申请的 imlsp.tk

Add 之后将会验证您的域名所有权，这里有四种验证方式

  • Meta Tag：在域名主页网站的 <head> 标签内增加 Yandex 所提供的代码行
  • HTML File：在网站根目录下创建 yandex_xxxxx.html 的文件并将文本框中的代码复制进去
  • DNS Record：在域名管理页面将文本框中的内容添加为TXT Record记录，正常情况下，最长需要 72h 才能完成验证
  • WHOIS：不推荐

这里我选择 DNS 验证

然后点击 Check 检查验证

OK，到这一步，我们直接就听它的，进行解析

如下图

说是说 72 小时，但有时生效也挺快的

接着我们进入 Yandex-admin

点击下面 Add 可以添加用户

输入信息，然后 Add

最多可以添加 1000 个，国外真是大方啊！

进入 Yandex 邮箱，登录刚刚创建的 lsp@imlsp.tk 域名邮箱

同意它的服务条款，不同意不行啊

到了这里就已经成功创建免费的域名邮箱啦

我们试试收发 email

可以看到已经正常收到了

发邮件也正常发送

在 Foxmail 添加 Yandex

一般有多种邮箱的话，管理起来是非常不好管理的，还要专门进到网页收发邮件

常见的邮箱管理有：

Foxmail 网易邮箱大师 畅邮 Outlook

等…

我个人喜欢用 Foxmail 管理所有邮箱服务

那么我就讲下如何在 Foxmail 添加 Yandex 邮箱

首先进入到 Yandex邮箱，点击右上角的齿轮 –> All settings

选择 Email clients，如图开启 POP3

打开 foxmail，新建账号

POP 服务器填 pop.yandex.com

SMTP 服务器填 smtp.yandex.com

最后注意要勾上 SSL，点击创建

成功创建
