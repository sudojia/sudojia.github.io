那些关于GitHub的谷歌插件

发表于2020-11-22|更新于2020-11-22|后端GitHub
|字数总计:444|阅读时长:1分钟

关于怎么访问 chrome 应用商店请自备科学的东西

当然你也可以在国内的插件市场下载

比如 Crx4Chrome，chajian5，Crx4…

Octotree

Octotree 就厉害啦，它可以在你的 GitHub 项目生成一个文件菜单树，就跟在 IDEA 中看到的效果一样。挺好用的，但是在Gitee不管用，如果你想在 GitHub 和 Gitee 同时有效的，也有一个插件，叫 GitCodeTree

你就说牛不牛逼，方不方便就 vans 了

GitHub 加速

这个插件据说能提高在国内访问 GitHub 的速度，还可以提高 clone 仓库的速度，提升下载 release 包的下载速度。

如果你正常安装的情况下，在 clone 的右边会有一个加速，以及在 release 会有个快速下载

之后 clone 的话，就用加速的 url

嗯~ o(￣▽￣)o 针不戳！！我这是手机热点，这速度，可以的啦

Awesome Autocomplete for GitHub

这个其实就是在 GitHub 的搜索栏又添加一个搜索功能，增强了 GitHub 的搜索功能

不过就是弄了个类型，仓库和用户类型。喜欢的可以下载

File Icons for GitHub and GitLab

这个插件就比较有趣啦，就是给文件加一个图标，不同的文件不同的图标

Enhanced GitHub

众所周知，GitHub 不能下载单个文件，这让人非常头疼呐！如果你就只想要下载一个文件的时候，那你就得需要 clone 整个项目。Enhanced GitHub 就可以对单个文件进行下载，太牛逼了，爱了爱了

不仅能下载单个文件，而且还可以看单个文件的大小，以及整个仓库的大小

先介绍到这了，后续还有啥好玩高效的再来补充！！
