折腾了三天，终于解决了使用多吉云首页打不开问题

发表于2024-04-19|更新于2024-04-19|经验分享
|字数总计:267|阅读时长:1分钟

起因

在使用多吉云的时候，打开我的部落格是这样的

这让我百思不得其解，打开控制台，发现是这个 CSS 加载不了

当我打开了 Proxy 去访问，结果又能访问到，所以一开始我的猜测就是会不会这个 CSS 没走多吉呢，而是走的 GitHub 直连。

然后就问了客服

客服就建议我在 CSS 后缀里面加个版本号试试，首页确实是可以打开，但是并没什么用，我其它的 XML，JS 等，反正就是我博客根目录的文件全被屏蔽了

后面问了 @瓦解哥 也是无解，最后换了个大胆的想法，就是会不会是 CDN 访问不了 GitHub 才导致的？

紧接着联想到 Vercel 有国内的 CNAME：cname-china.vercel.com

最后才因此解决这个鬼问题！

心累，刚回来折腾一下部落格就碰到这个鬼问题！

总结

就是因为 GitHub 被墙了，多吉的 CDN 回源的过程中产生超时！
